# Tenkići
Autor projekta je Ljubiša Đikić. Cilj projekta je simulacija kosog hica na primeru igrice sa dva tenka koji se pucaju.
**Ne menjajte veličinu blokova, to će sve da pokvari!**

## Kontrole:
| Dugme | Akcija |
| ----------- | ----------- |
| **a** | levo |
| **d** | desno | 
| ↑ | nišani više |
| ↓ | nišani niže |
| **w** | povećava snagu |
| **s** | smanjuje snagu |
| **space**| puca |


## Potrebne biblioteke

- `numpy`
